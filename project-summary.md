# Project Summary

## Project Name

CareerPilot AI

## Track

Concierge Agents

## Problem Statement

Students often struggle with resume building, internship discovery, career planning, and interview preparation.

## Solution

CareerPilot AI provides an intelligent AI-powered assistant that helps students navigate their career journey through personalized recommendations and guidance.

## Key Features

- Resume Analysis
- Career Guidance
- Internship Recommendations
- Learning Roadmaps
- Interview Preparation

## AI Concepts Used

- Agent Development Kit (ADK)
- MCP Concepts
- Agent Skills
- Security Guardrails

## Impact

CareerPilot AI helps students make informed career decisions and improve employability through personalized AI assistance.
