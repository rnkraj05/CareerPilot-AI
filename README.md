# CareerPilot AI

An AI-powered Career Assistant designed to help students, internship seekers, and early professionals make informed career decisions.

## Features

- Resume Analysis & Feedback
- Internship Recommendations
- Career Guidance
- Personalized Learning Roadmaps
- Interview Preparation Assistance
- AI-powered Career Planning

## Technologies Used

- React
- TypeScript
- Vite
- AI Agent Concepts
- MCP Integration Concepts
- Agent Skills Architecture

## AI Agent Concepts Demonstrated

- Agent Development Kit (ADK)
- Model Context Protocol (MCP)
- Agent Skills
- Security Guardrails
- Tool-based Agent Workflow

## Repository Structure

src/ - Application source code  
public/ - Static assets  
app/ - Application resources  
README.md - Project documentation
